RPDB Integrations

This page describes the many ways in which RPDB can be used in combination with various apps: Plex, Emby, Jellyfin, Kodi, Stremio and more.

Plex

Plex can currently use RPDB posters through the RPDB Folders app or the RPDB Media Agent. See the RPDB Plex Guide.

Emby

Emby can currently use RPDB posters through the RPDB Folders app or the Emby Plugin. See the RPDB Emby Guide.

Jellyfin

Jellyfin can currently use RPDB posters through the RPDB Folders app or the Jellyfin Plugin. See the RPDB Jellyfin Guide.

Stremio

Stremio can currently use RPDB posters with over 8 addons, see the RPDB Stremio Guide.

Kodi

Kodi can currently use RPDB posters (only for local media) through the RPDB Folders app. See the RPDB Kodi Guide.

Web Panel

The RPDB Web Panel can be used to edit and download posters and backdrops online through our website. You can access the Web Panel by going to this link.
